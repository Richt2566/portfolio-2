PROJECTS

  • ChicagoLive
  • Game ON!
  • onDeck
  • Death Metal Hangman

Chicago Live was created for the love of live music. It was written in HTML, CSS and the Javascript library jQuery. The search function queries the Ticketmaster API. We wanted to pay tribute to applications we love like bandsintown. I was the lead developer in a group of four.

Game ON! is an easy to use website reservation system for sports enthusiasts to create games with each other. It was written in HTML, CSS, and Javascript. The backend was written in Node.js, Express, and Handlebars.js with a mySQL database.

onDeck is an intricate React application that replaces a physical deck of cards. Users have the ability to sign in, shuffle, draw, discard, and pass their turn to an opponent. The frontend was written in Reactstrap with a firebase database.

This simple game was a demonstration of the power of Javascript and to get familiar with one aspect of the complexities of the language. It's almost as complex as the wicked riffs coming from Nile's Annihilation of the Wicked.
